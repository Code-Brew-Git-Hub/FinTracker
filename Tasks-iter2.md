### Backend-разработчик
- Спроектировать инфраструктуру для ASP.NET Сервиса (REST-API)
- Реализовать контроллер, который принимает файлы с транзакциями

### Frontend-разработчик
- Сверстать страничку, на которой загружается файл в сервис

### DevSecOps-инженер
- Проектирование базы данных (предметной области) для хранения данных нашего сервиса (транзакции, категории и тд)
- Реализовать модуль работы с бд (сохранение, получение, создание бд и тд) в проекте FinTracker.Data. База данных должна быть локальной (сохраняться локально)

### UX/UI-проектировщик
- Создать дизайн странички на которой мы загружаем файл в сервис


### Системный аналитик
- Исправить замечания из pull request
- Создать документ, подробно описывающий модуль агрегации (его работа, требования к нему, примеры работы)
